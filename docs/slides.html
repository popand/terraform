FortiGate Day-0 Configuration

Terraform Template

user_data = templatefile( "fortigate_config.tpl", { hostname = var.name admin_password = var.admin_password private_ip = aws_network_interface.private.private_ip private_netmask = "255.255.255.0" vpn_name = var.vpn_name vpn_peer_ip = var.vpn_peer_ip vpn_psk = var.vpn_psk local_subnet = var.private_subnet_cidr remote_subnet = var.remote_private_subnet_cidr })

Key Features

  • Dual-NIC: port1 (public), port2 (private)
  • DHCP on public, static on private
  • Firewall address objects for subnets
  • Static route to remote via VPN tunnel

Generated FortiOS Config

# System & Interfaces config system global set hostname "FortiGate-1" config system interface edit "port1" set mode dhcp set allowaccess ping https ssh edit "port2" set ip 10.0.1.254 255.255.255.0 # IPSec VPN Phase 1 & 2 config vpn ipsec phase1-interface edit "vpn-to-vpc2" set remote-gw 3.147.113.4 set proposal aes256-sha256 set psksecret ******** config vpn ipsec phase2-interface set src-subnet 10.0.1.0/24 set dst-subnet 10.100.1.0/24 # Firewall Policies (3 rules) vpn-outbound: port2 → vpn-to-vpc2 vpn-inbound: vpn-to-vpc2 → port2 outbound-nat: port2 → port1 (NAT)
